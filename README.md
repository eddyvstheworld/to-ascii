## A Vue and Flask app, made to take you back to a simpler time.


## Requirements
To properly run this app, you'll need to be running Python 3.6 or above installed in your system. Additionally, you'll need either NPM or Yarn for the front end.

## Install
To install and get the server running, install the packages with pip (preferably with a virtual environment):
```
  cd server
  pip install -r requirements.txt
  python toascii.py
```

As for the front end:
```
  cd client
  yarn install
  yarn run serve
```
